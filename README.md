### supervisor的操作:
```
[root@****** ~]# ps -ef|grep supervisor
[root@****** ~]# kill +进程
[root@****** ~]# supervisord -c /etc/supervisord.conf
[root@****** ~]# ps -ef|grep supervisor
[root@****** ~]# supervisorctl update
[root@****** ~]# supervisorctl
```
